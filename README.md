# MCP-Summarizer

A multi-format document summarizer using **Model Context Protocol (MCP)** and LLMs.  

## Features

- Supports text files, PDFs, scanned PDFs, and images
- Automatic tool selection using MCP
- Chunked summarization for long documents
- Structured output: Key Points, Short Summary, Main Theme
- CLI interface with ASCII splash screen
- Lightweight and CPU-friendly (no heavy GPU needed)

## Usage

```bash
python run_mcp.py
